= pnm

Image library in pure Nim.

image:https://travis-ci.org/jiro4989/pnm.svg?branch=master["Build Status", link="https://travis-ci.org/jiro4989/pnm"]

WARN:: Developing now.

== Usage

[source,nim]
----
import pnm

block:
    let f = open("p1.pbm")
    let pbm = f.readPBM()
    echo pbm

block:
    let pbm = f.readPBMFile("p4.pbm")
    echo pbm

----
